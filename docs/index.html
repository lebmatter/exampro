Everything you need

From quick classroom quizzes to high-stakes proctored exams — create, conduct, and evaluate with one app.

Quick Quiz

Run gamified, real-time quizzes with Kahoot-style competitive mode or simple standalone quizzes.

  • Kahoot & simple quiz modes
  • PIN-protected or authenticated access
  • Live leaderboard & score bonuses
  • Multiple themes & question timers

AI-Powered Content

Generate questions, help text, and images using AI. Supports multiple models via OpenRouter.

  • AI question generation from prompts
  • Contextual help text & tutoring
  • AI image generation for questions
  • Configurable difficulty levels

Studio Editor

A visual editor for building exams and quizzes with drag-and-drop simplicity and batch import.

  • Visual question builder
  • Batch import from question bank
  • Category search & filtering
  • State caching for work-in-progress

Live Proctoring

Monitor candidates in real-time with webcam surveillance, face detection, and automatic recording.

  • Real-time video monitoring
  • Face detection & multi-face alerts
  • Tab & window focus tracking
  • Configurable warning thresholds

Flexible Questions

Multiple choice, free text, image-based questions with explanations, help quizzes, and difficulty levels.

  • MCQ & subjective question types
  • Image-based questions & options
  • Embedded help quizzes per question
  • Question bank with categories

Certificates & Leaderboards

Auto-generate digital certificates on completion and display competitive leaderboards.

  • Custom certificate templates
  • Configurable certificate expiry
  • Exam & schedule-level leaderboards
  • Time-based ranking
